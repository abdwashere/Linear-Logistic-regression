linear.py  (Interactive Linear Regression Canvas)
Click anywhere on a canvas to drop data points. The model instantly retrains and redraws the best-fit line after every click. Built with tkinter + scikit-learn.
Great for understanding how linear regression responds to new data points in real time.

studentmarkprediction.py (Student Mark Predictor)
Predicts a student's marks based on input features using a linear regression model.

University_Admission.py  (University Admission Classifier)
Uses logistic regression to predict whether a student is likely to get admitted based on their profile.

install dependencies 
pip install scikit-learn numpy matplotlib
